# 📊 ElectViz – Election Data Visualization for Media

ElectViz is an interactive Power BI project designed to analyze election results across multiple years and present them in a structured, media-style dashboard format.

This project transforms raw election data into meaningful visual insights using data modeling, DAX calculations, and interactive visualizations.

---

## 📌 Project Objectives

- Analyze historical election trends across 2009, 2014, and 2019
- Simulate a real-time election results dashboard
- Identify leading parties based on constituencies won
- Visualize seat distribution and vote share
- Analyze voter turnout and regional participation patterns

---

## 🗂 Dataset Description

The dataset includes election results for three years (2009, 2014, 2019) with the following fields:

- Year
- Constituency
- Candidate
- Party
- Votes
- Turnout %
- Total Electors

---

## 📊 Dashboards Overview

### 1️⃣ Historical Election Trends Dashboard

This dashboard focuses on long-term analysis across multiple elections.

**Key Visuals:**
- Voting & Turnout Trends Over Time
- Party Vote Comparison by Year
- Constituency-wise Turnout Analysis
- Top 5 Constituencies by Vote Growth
- Geographical Vote Distribution Map
- Average Turnout by Year

**Purpose:**
To identify patterns, participation changes, and shifts in party performance over time.

---

### 2️⃣ Real-Time Results Dashboard

This dashboard simulates a live election reporting interface using final result data.

**Key Features:**
- Leading Party (based on constituencies won)
- Majority Mark Calculation
- Seat Distribution by Party
- Seat Share Distribution (Pie Chart)
- Total Votes & Seats Won KPIs
- Winning Candidates Table
- Interactive Year Slicer

**Purpose:**
To provide a real-time style summary of election outcomes for a selected year.

---

## 🧮 Key DAX Calculations

- Rank candidates within each constituency
- Calculate Seats Won per party
- Determine Leading Party dynamically
- Compute Majority Mark threshold
- Calculate Vote Growth %
